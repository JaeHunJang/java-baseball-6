### 2. 사용자가 입력한 3자리 수 비교하기
    - camp.nextstep.edu.missionutils에서 제공하는 Console을 사용하여 구현
    
### 3. 입력에 대한 결과 출력하기
    - 예) 상대방(컴퓨터)의 수가 425일 때

        456을 제시한 경우 : 1볼 1스트라이크
        789를 제시한 경우 : 낫싱

### 4. 숫자를 전부 맞추면 다시 시작하거나 종료하기
    - 게임 재시작은 1, 종료는 2
